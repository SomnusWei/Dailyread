# 穴位数据导入工具使用说明

## 文件说明

- `acupoint_converter.py` - 主程序文件
- `运行穴位工具.bat` - Windows一键启动脚本
- `requirements.txt` - Python依赖列表

## 快速开始

### 方法一：使用一键启动脚本（推荐）

1. 双击 `运行穴位工具.bat`
2. 等待依赖安装完成（首次运行时）
3. 程序界面将自动打开

### 方法二：手动运行

1. 确保已安装 Python 3.7 或更高版本
2. 安装依赖：
   ```bash
   pip install pandas openpyxl
   ```
3. 运行程序：
   ```bash
   python acupoint_converter.py
   ```

## 使用步骤

### 1. 导出Excel模板

1. 点击"导出Excel模板"按钮
2. 选择保存位置
3. 模板将包含示例数据（合谷、足三里）供参考

### 2. 填写Excel数据

在Excel中填写穴位信息，包含以下字段：

| 字段名 | 必填 | 说明 |
|--------|------|------|
| 穴位 | ✅ | 穴位名称（唯一标识） |
| 经络 | ✅ | 所属经络 |
| 穴性 | | 穴位性质 |
| 定位 | | 穴位位置描述 |
| 功效 | | 功效说明 |
| 主治 | | 主治病症 |
| 解剖 | | 解剖位置 |
| 操作 | | 针灸操作方法 |
| 禁忌 | | 禁忌事项 |
| 穴位定位图路径 | | 图片文件名（留空即可） |
| 备注 | | 其他备注信息 |

### 3. 转换为JSON

1. 点击"浏览..."选择填写好的Excel文件
2. 设置起始ID（首次使用可保持为1）
3. 点击"转换为JSON"按钮
4. 选择保存位置

**关于起始ID**：
- 如果是第一次导入：从1开始即可
- 如果已经有数据：查看APP中穴位的最大ID，设置为比最大ID大的值

### 4. 导入到APP

1. 将生成的JSON文件传输到手机
2. 打开DailyRead APP
3. 进入设置 → 导入数据
4. 选择生成的JSON文件

## Excel模板示例

```
穴位    经络          穴性    ...
合谷    手阳明大肠经    原穴    ...
足三里  足阳明胃经    合穴    ...
```

## 注意事项

1. **必填项**：穴位和经络为必填项，缺少会跳过该行
2. **图片路径**：目前不需要填写，APP内会单独处理
3. **ID设置**：避免与已有数据ID重复
4. **数据格式**：确保Excel使用标准格式，不要合并单元格

## 常见问题

### Q: 提示缺少Python怎么办？
A: 访问 https://www.python.org/downloads/ 下载安装，安装时勾选"Add Python to PATH"

### Q: 依赖安装失败怎么办？
A: 手动执行：`pip install pandas openpyxl`

### Q: 导入后穴位不显示？
A: 检查穴位和经络是否都有填写，这两项为必填

### Q: 如何批量导入多个文件？
A: 可以将多个Excel文件合并为一个，或者分别转换并导入（注意设置不同的起始ID范围）

## 技术支持

如遇问题，请检查：
1. Python版本（3.7+）
2. Excel文件格式（.xlsx格式）
3. 必填字段是否完整
4. JSON文件是否正确生成
